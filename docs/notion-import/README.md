## 🚀 Notionへのインポート手順

### 方法1: 個別ページとしてインポート（推奨）

1. **Notionを開く**
   - ブラウザで https://www.notion.so にアクセス
   - または Notionデスクトップアプリを起動

2. **新しいページを作成**
   - 左サイドバーで「+ 新規ページ」をクリック
   - ページ名を「S×S Manager」に設定

3. **ファイルをインポート**
   - ページ内で右上の「⋯」（その他）をクリック
   - 「インポート」→ 「Markdown」を選択
   - `00_PROJECT_HOME.md` を選択してインポート
   - 同様に `01_DATA_MODEL_DESIGN.md` から `09_IDEAS.md` まで順番にインポート

4. **階層構造を整理**
   ```
   S×S Manager (00_PROJECT_HOME)
   ├── 📚 設計書
   │   ├── 01_データモデル設計書
   │   ├── 02_API仕様書
   │   ├── 03_画面遷移図・機能一覧
   │   ├── 04_ビジネスロジック仕様書
   │   ├── 05_技術アーキテクチャ設計書
   │   └── 06_管理者機能設計書
   ├── 🗂️ プロジェクト管理
   │   ├── 07_タスク管理
   │   └── 08_スケジュール
   └── 💡 アイデア
       └── 09_アイデアメモ
   ```

5. **ページをドラッグ&ドロップで整理**
   - サイドバーでページをドラッグして階層構造を作成

---

### 方法2: 一括インポート

1. **Notionを開く**

2. **インポート画面へ**
   - 左サイドバー下部の「設定」→「インポート」
   - 「Markdown」を選択

3. **フォルダごとインポート**
   - このディレクトリ全体（`docs/notion-import/`）を選択
   - Notionが自動的に階層構造を作成

## ⚠️ 注意事項

### Markdownの制限

Notionは標準Markdownとは一部異なります:

**サポートされているもの:**
- ✅ 見出し（# ## ###）
- ✅ リスト（- * 1.）
- ✅ チェックボックス（- [ ]）
- ✅ コードブロック（```）
- ✅ テーブル
- ✅ リンク

**サポートされていないもの:**
- ❌ 複雑なHTMLタグ
- ❌ カスタムCSS
- ❌ 一部の絵文字（自動変換されます）

### インポート時の注意

- ファイルサイズ: 最大5MB
- 一度に大量のファイルをインポートすると時間がかかる場合があります
- 日本語ファイル名は正しく表示されます

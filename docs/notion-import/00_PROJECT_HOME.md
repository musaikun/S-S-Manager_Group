# S×S Manager プロジェクト

> 複数のアルバイト先を掛け持ちする従業員向けのシフト管理アプリケーション

**最終更新:** 2026-01-09
**プロジェクトオーナー:** 無才
**バージョン:** 1.0.0

---

## 🎯 プロジェクト概要

### ビジョン
誰でも使いやすい、直感的なシフト管理アプリケーションの提供

### 対象ユーザー
- **従業員**: シフト希望を簡単に提出
- **管理者**: シフト承認・集計・分析

### 主要機能
1. カレンダー選択（直感的なUI）
2. 掛け持ち管理（最大4つのジョブ）
3. 時間設定（一括 & 個別）
4. 競合検出（時間重複の自動検出）
5. 統計表示（勤務日数・時間）
6. **🆕 シフト確認**（提出済みシフトの閲覧）
7. **🚧 管理者ダッシュボード**（開発予定）

## 🛠️ 技術スタック

### 現在の構成
```
Frontend: Vue 3 + TypeScript + Vite + Pinia
Backend:  Spring Boot 3.5.7 + Java 21（基本構成のみ）
```
